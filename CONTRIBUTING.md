# Contributing to Avatar Fight

Thanks for testing! This file explains the easiest ways to give feedback.

## Filing a bug

1. Open an issue with the **Bug report** template
2. Tell us what device / iOS version
3. Describe what you expected vs. what happened
4. Screenshots / screen recordings are gold — drag them right into the issue

## Requesting a feature

1. Open an issue with the **Feature request** template
2. Describe the feature in 1–2 sentences
3. Tell us *why* it would matter to you — the use case is more useful than the spec

## Discussions

For open-ended ideas ("what if pets had stories?") use [Discussions](https://github.com/Uwais12/avatar-fight/discussions). Issues are for actionable items.

## Code contributions

Solo project for now — code PRs are paused while the architecture stabilises. Stars, issues, and feature requests are the contributions that matter most right now.
